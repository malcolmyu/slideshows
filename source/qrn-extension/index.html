页面跳转 API

  • open(name[, opts])打开一个新的页面（新建历史）
  • back([opts])回到上一个页面
  • backTo(name[, opts])回到指定页面
  • goto(name[, opts])前往指定页面（历史中有则回退）
  • home([opts])回到首页
  • close(name)关闭指定页面
